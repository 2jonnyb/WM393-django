Instructions to run.

1. Create a python virtual environment.
2. Install requirements.txt
3. In the environment, run WMGTSS/manage.py runserver
4. In a browser, connect to localhost:8000 or 127.0.0.1:8000
5. To use the admin page, go to localhost:8000/admin
